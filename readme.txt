This simple to use plugin will change the standard Admin Greeting from 'Howdy' to the Maori version 'Kia Ora'.

== Description ==

If you're like us at ROIMarketing.co.nz and are sick of being greeted to your Wordpress Admin panel with a Texas greeting of 'Howdy' then you'll love this plugin.

Simply install this plugin and you will instantly be greeted with the Maori greeting of Kia Ora and the current New Zealand flag - a little bit of Kiwiana inside your Wordpress dashboard.

More Info: [ROI Marketing](http://www.roimarketing.co.nz/articles/plugin-change-your-admin-bar-greeting-to-maori/)

== Installation ==

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
